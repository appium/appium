# Scroll

Scroll on the touch screen using finger based motion events
[//]: # (DO NOT EDIT THIS FILE! This is an auto-generated file. Editing for this document happens in /commands-yml/commands/interactions/touch/scroll.yml)
## Example Usage

```java
// Java
TouchActions action = new TouchActions(driver);
action.scroll(element, 10, 100);
action.perform();

```

```python
# Python
from appium.webdriver.common.touch_action import TouchAction
# ...
actions = TouchAction(driver)
actions.scroll_from_element(element, 10, 100)
actions.scroll(10, 100)
actions.perform()

```

```javascript
// Javascript
// webdriver.io example
driver.touchScroll(10, 100, element);

// wd example
await driver.scroll(10, 100);

```

```ruby
# Ruby
# ruby_lib example
touch_actions.scroll(element, 10, 100).perform

# ruby_lib_core example
@driver.touch_actions.scroll(element, 10, 100).perform

```

```php

```

