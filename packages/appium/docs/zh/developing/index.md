title: 开发 Appium 扩展
---

Appium 采用模块化结构，这意味着 Appium 扩展（驱动和插件）与主 Appium 模块是解耦的；你只需要安装自己想要使用的扩展即可。 这种模块化结构还使得开发全新的扩展变得成为可能。

Appium 文档的这一部分，旨在帮助有志于开发者创建自己的 Appium 扩展：

- 想要创建驱动，请查看 [构建驱动](./build-drivers.md) 页面
- 想要创建插件，请查看 [构建插件](build-plugins.md) 页面
- 驱动和插件都需要文档，因此请查看 [构建文档](./build-docs.md) 页面
- 想要创建 Doctor check，请查看 [构建Doctor Checks](./build-doctor-checks.md) 页面
