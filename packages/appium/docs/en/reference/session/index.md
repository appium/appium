title: Session Properties
---

An Appium session has several key properties, each with its own set of supported values:
capabilities, execute methods, settings, locator strategies, and others. The supported values for
each property are primarily defined by the active driver and plugins, but some properties also have
common values provided by the Appium server and its base driver:

    A small set of basic capabilities

-   [__Command Endpoints__](../api/index.md)

    The core set of API endpoints
