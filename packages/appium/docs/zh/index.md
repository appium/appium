欢迎访问 Appium 文档！Appium 是一个开源项目和相关软件的生态系统。
软件，旨在促进许多应用程序平台的用户界面自动化，包括移动平台（iOS、
Android、Tizen）、浏览器（Chrome、Firefox、Safari）、桌面（macOS、Windows）、电视（Roku、tvOS、
Android TV、三星）等！
